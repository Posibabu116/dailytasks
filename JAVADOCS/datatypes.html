JavaScript Data Types

JavaScript provides different data types to hold different types of values. Here are the main data types:

Number

A number is used to represent numeric values. JavaScript has only one type of number, which can be written with or without decimals.

// Example of a number
let age = 25;
let pi = 3.14;
console.log(age); // Output: 25
console.log(pi); // Output: 3.14

Null

Null is a special value that represents "no value" or "nothing".

// Example of null
let emptyValue = null;
console.log(emptyValue); // Output: null

String

A string is a sequence of characters used to represent text.

// Example of a string
let name = "John Doe";
console.log(name); // Output: John Doe

BigInt

BigInt is a special numeric type that provides support for integers of arbitrary length.

// Example of BigInt
let bigNumber = 1234567890123456789012345678901234567890n;
console.log(bigNumber); // Output: 1234567890123456789012345678901234567890n

Boolean

A boolean represents one of two values: true or false.

// Example of boolean
let isStudent = true;
let hasGraduated = false;
console.log(isStudent); // Output: true
console.log(hasGraduated); // Output: false

Symbol

Symbol is a unique and immutable primitive value and may be used as the key of an object property.

// Example of Symbol
let symbol1 = Symbol("description");
let symbol2 = Symbol("description");
console.log(symbol1 === symbol2); // Output: false

Undefined

A variable that has not been assigned a value has the value undefined.

// Example of undefined
let notAssigned;
console.log(notAssigned); // Output: undefined

Object

An object is a complex data type that allows you to store collections of data.

// Example of an object
let person = {
    firstName: "John",
    lastName: "Doe",
    age: 30,
    isStudent: true
};
console.log(person);
console.log(person.firstName); // Output: John

Array

An array is a special type of object used to store multiple values in a single variable.

// Example of an array
let colors = ["red", "green", "blue"];
console.log(colors);
console.log(colors[0]); // Output: red
