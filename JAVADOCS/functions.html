JavaScript Functions Examples

1. Named Function

Name: Named Function

Definition: A named function is defined using the `function` keyword and can be called by its name.


Syntax:
function functionName(parameters) {
    // Function body
    return value; // Optional
}

2. Function with Arguments

Name: Function with Arguments

Definition: A function that accepts parameters (arguments) which are passed during its invocation.


Syntax:
function functionName(parameter1, parameter2, ...) {
    // Function body
    return value; // Optional
}

Example:

3. Asynchronous Function

Name: Asynchronous Function

Definition: An asynchronous function is defined using the `async` keyword and can use `await` within its body to handle asynchronous operations.


Syntax:
async function functionName() {
    try {
        let result = await asyncOperation();
        return result;
    } catch (error) {
        console.error('Error:', error);
    }
}

Example:

4. Immediately Invoked Function Expression (IIFE)

Name: Immediately Invoked Function Expression (IIFE)

Definition: An IIFE is a function that is executed immediately after it's defined.


Syntax:
(function() {
    // Function body
})();

Example:

5. Callback Function

Name: Callback Function

Definition: A callback function is a function passed as an argument to another function and is executed when a specific event occurs.


Syntax:
function functionName(callback) {
    // Function body
    callback();
}

Example:

6. Arrow Function

Name: Arrow Function

Definition: An arrow function is a concise way to write function expressions in JavaScript.


Syntax:
const functionName = (parameter1, parameter2, ...) => {
    // Function body
    return value; // Optional
};

Additional Examples

1. Arithmetic Operations Function

Name: Arithmetic Operations Function

Definition: A function that performs basic arithmetic operations (+, -, *, /).


Syntax:
function arithmetic(a, b) {
    let result = "";
    result += "Addition: " + (a + b) + "\n";
    result += "Subtraction: " + (a - b) + "\n";
    result += "Multiplication: " + (a * b) + "\n";
    result += "Division: " + (a / b) + "\n";
    return result;
}

3. Maximum of Two Numbers Using Ternary Operator

Name: Maximum of Two Numbers Using Ternary Operator

Definition: A function that finds the maximum of two numbers using the ternary operator.


Syntax:
function max(a, b) {
    return (a > b) ? a : b;
}

4. Calculate Total Interest Function

Name: Calculate Total Interest Function

Definition: A function that computes the total interest based on the principal amount, rate of interest, and number of years.


    Syntax:
    function calculateInterest(amount, rate, years) {
        let interest = (amount * rate * years) / 100;
        return interest;
    }
