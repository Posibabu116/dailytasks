1. How isNaN Works in Different Scenarios

The isNaN function determines whether a value is NaN (Not-a-Number). It returns true if the value is NaN, and false otherwise.

// Examples of isNaN
isNaN(NaN);        // true
isNaN(123);        // false
isNaN('123');      // false
isNaN('Hello');    // true
isNaN(undefined);  // true
isNaN(null);       // false
isNaN(true);       // false
isNaN(false);      // false

Note: isNaN coerces the value to a number before checking if it's NaN.

2. Types of Coercion in JavaScript

JavaScript uses two types of coercion:

  • Implicit Coercion: Automatically performed by JavaScript. It occurs in expressions involving different types.
  • Explicit Coercion: Manually performed by the developer using functions or operators.

Implicit Coercion

// Examples of implicit coercion
let result1 = '5' + 5;     // '55' (string)
let result2 = '5' * 5;     // 25 (number)
let result3 = '5' - 3;     // 2 (number)
let result4 = 5 + true;    // 6 (number)
let result5 = 5 + null;    // 5 (number)

Explicit Coercion

// Examples of explicit coercion
let num = Number('5');    // 5 (number)
let str = String(5);      // '5' (string)
let bool = Boolean(1);    // true (boolean)

3. Program to Print a Random OTP Using Math Methods

The following JavaScript program generates a random 4-digit OTP:

// Function to generate a random 4-digit OTP
function otpgen() {
    var otp = 1000 + Math.floor(Math.random() * 9000);
    console.log(otp);
}

// Generate and print the OTP
otpgen();
